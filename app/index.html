Blockstack.
The Open Source Blockchain Community

Our mission is to create common tools for building decentralized applications. We welcome all developers to join the Blockstack community and contribute to open source application and infrastructure development.

Decentralized

Applications can be built without central points of failure.

Open Source

Blockstack code is free to inspect, modify, and reuse.

Community

We are developers united around a common software stack.

Use Blockstack to build applications faster and with less deployment costs.

Messaging

Use p2p protocols to send and receive messages.

Naming

Use the Bitcoin blockchain to create secure identifiers.

Storage

Use the blockchain and a DHT or HTTPS for data storage and retrieval.

Consensus

Use the Bitcoin protocol to keep clients in sync.
